# Contributing to Voice Skill Generator

Thanks for being here. This is a small, focused repo and contributions should match that energy — useful, specific, and not bloated.

---

## Ways to contribute

### Share a voice skill you built
If you ran the generator and built a skill you're happy with, consider opening a PR to add it to the `example/` folder. Include:
- A folder named `your-name-or-handle-voice/`
- Your `SKILL.md` inside it
- An optional `README.md` in the folder describing what content types it covers

### Improve the generation prompt
The `USAGE_PROMPT.md` is the core of this repo. If you found a way to get better output — better section structure, better analysis questions, clearer instructions — open an issue describing what you changed and what improved.

### Report a bad output
If the generator produced something that clearly didn't match your voice, open an issue with:
- What writing samples you used (length, type)
- What felt off in the output
- What you expected instead

Not asking you to paste your personal writing publicly — just describe the failure mode.

### Fix something broken
Typos, broken links, unclear instructions — open a PR. No need to file an issue first for small fixes.

---

## What this repo is not looking for

- New features for the sake of features
- Automated tooling, scripts, or CI/CD additions
- Voice skills for AI-generated content (they won't work well and they'll pollute the examples folder)
- Rewrites of the README in a different style

---

## How to open a PR

1. Fork the repo
2. Make your changes in a branch
3. Open a PR with a clear title and a one-paragraph description of what you changed and why
4. That's it — no templates, no lengthy checklists
